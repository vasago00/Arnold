What changed & why

Same engine, same brand — re‑sequenced so the screen answers what · why · how‑sure in one glance, the thing every competitor (WHOOP, Oura, even with Oura's own advisor calling the scores "made up") fails to do.

1

Transparency is the hero

State + the top 2–3 weighted contributors + a confidence bar, from the existing attribution.js / hubFacts / LearnedHero. This is the demo that wins — Athlytic markets "why your score is what it is"; Arnold can show it deeper.

2

One hero per screen

Working memory tops out at ~3–5 items; cluttered dashboards measured 12 min→insight vs 3 min optimized. The 12‑tile EdgeIQ rail collapses to hero + a few + "More."

3

Progressive disclosure (≤2 levels)

Summary → trend → raw. "More drivers", "open ↗", and "tap to expand" defer detail instead of stacking it — NN/G's key mobile guideline.

4

One Coach voice + prescriptive fuel

The teal serif Coach speaks once about what changed (already best practice). The "fuel for tomorrow" card + EA flag finally makes the Cronometer data pay off — MAVR/Fuelin's whitespace.

5

Color does one job, paired with shape

Green/amber/red = status only; the ＋ / － / ≈ glyphs carry the same meaning for accessibility. Brand/category accents stay separate (tokens already encode this).
