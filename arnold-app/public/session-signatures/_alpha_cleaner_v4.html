Alpha Cleaner v4 flood-fill

What's different in v4. The previous tool deleted any pixel within color-distance of the corners — which could eat figure pixels that happened to match the background. v4 uses flood-fill from edges + click-as-magic-wand to preserve figure integrity. Pixels are only removed if they're reachable through a chain of similar-color neighbors from either an image edge or a point you clicked. Isolated figure pixels that happen to match background colors stay intact.

How to use.
1. Drop one or more PNGs. The 4 corner colors are auto-sampled.
2. Click on any leftover background patch in the ORIGINAL preview — it acts as a magic-wand seed. The flood-fill expands outward from that exact pixel, catching isolated background islands that aren't connected to the image edge (this is the common Gemini failure mode where checkerboard patches persist inside the image).
3. Repeat for each isolated patch. Each click adds another flood-fill starting point.
4. Tune color tolerance if too much/too little gets caught. Raise to clean stubborn anti-aliased edges; lower if figure pixels get nibbled.
5. Hit Reset samples if you misclicked and want to start over.
