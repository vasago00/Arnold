NaN / Undefined Guard Pattern (Phase E)

MobileHome tile display pattern:
1. Compute raw value → fallback || 0
2. Filter array: .filter(v => typeof v === 'number' && !isNaN(v))
3. Final display: isNaN(avg) ? '—' : avg.toFixed(0)
4. Gauge percentage clamp: (isNaN(gaugePct) || !isFinite(gaugePct)) ? 0 : Math.max(0, Math.min(gaugePct, 1))
Prevents NaN crashes when Garmin data missing, date-window empty, or arithmetic divides by 0. Applied to every metric tile in Run/Strength/Recovery/Body categories.
